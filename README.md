# vue2-template

```sh
* vite+element-ui+axios+scss开箱即用
* git clone url
* git remote remove origin
* 不要提交代码
```

## 环境
```sh
在env编写所需环境变量，使用 import.meta.env访问
```
## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
